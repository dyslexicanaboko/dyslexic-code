This directory contains example code for a DaisyLink V4 compatible Multicolor LED module.
This code targets a Cypress Semiconductor PSoC CY8C21223-24LGXI programmable logic device.

To edit and build this code, we use PSoC Designer 5.1 from Cypress Semiconductor.
To examine the reference implementation of DaisyLink, look in the DaisyLink.c file.
